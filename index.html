HMS - Harmful Brain Activity Classification and Analysis

An advanced project for detecting and classifying harmful brain activities using EEG signals.

About the Project

EEG signals provide non-invasive insights into brain activity. Our project aims to detect seizures and other harmful patterns using advanced signal processing and machine learning techniques.

Objectives

  • Observe and categorize the pattern of EEG data.
  • Detect significant anomalies in the data patterns.
  • Train a machine learning model on selected datasets.
  • Test the model using third-party datasets for robustness.
  • Develop a web and mobile application for easy use.

Dataset & Methodology

We use open-source EEG datasets and employ advanced pre-processing and feature extraction techniques such as wavelet transformation, filtering, etc.

Diseases & Their Effects on the Brain

Many neurological diseases significantly alter brain activity. Below are some of the diseases that we aim to analyze using EEG signals:

  • Epilepsy: Characterized by recurrent seizures, epilepsy alters the brain's electrical patterns, which can be identified using EEG data.
  • Parkinson's Disease: A degenerative disorder affecting movement, often associated with abnormal brain wave activity in certain regions.
  • Alzheimerâ€™s Disease: A progressive disorder affecting memory and cognitive function, leading to abnormal EEG patterns.
  • Stroke: Damage to brain cells from a stroke can alter brain activity, which may show as irregularities in EEG data.
  • Sleep Disorders: Conditions such as insomnia or sleep apnea can be studied through abnormal brain wave patterns during sleep cycles.

Application Demo

Our web-based application provides real-time classification of brain activity using pre-trained models. You can download the mobile app or use the web interface.
